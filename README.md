# fluent-plugin-chatter-api

Fluentd input plugin for chatter api.

## Installation

Add this line to your application's Gemfile:

    gem 'fluent-plugin-chatter-api'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fluent-plugin-chatter-api

## Usage


### Fluentd config

    <source>
      type chatter-api
    </source>
